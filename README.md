# 🐍 Snake Water Gun Game (Python)

## 📌 Overview
This is a simple Snake Water Gun game built using Python. The player competes against the computer, and the winner is decided based on the game rules.

## ✨ Features
- Play against the computer
- Random computer choice using the random module
- User-friendly console input
- Win, Lose, and Draw conditions

## 🛠️ Technologies Used
- Python 3
- Random Module

## ▶️ How to Run
1. Download or clone this repository.
2. Open the project in VS Code.
3. Run:
   bash
   python main_project.py
